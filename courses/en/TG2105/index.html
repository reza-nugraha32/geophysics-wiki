TG2105

Computing in Geophysics

Course Description

This course is a coding-oriented numerical methods course designated for second year geophysics students. It covers topics in numerical methods including system of linear equations, curve fitting, numerical integration, numerical differentiation, differential equation, and optimizations. This course is taught using Python.

Prerequisites

Introduction to Computation
