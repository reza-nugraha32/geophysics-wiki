Frequently Asked Question (FAQs)

Below are frequently asked questions about OpenGeophy.

+ How to use OpenGeophy?

Open OpenGeophy contains courses materials from the geophysical engineering's undergraduate curriculum in Institut Teknologi Bandung, Indonesia. To use OpenGeophy, simply search for the course/subject that you want to learn and use the reading materials, problem sets, or the past papers available.
Note that OpenGeophy can't be a substitute for a textbook. Please use a proper textbook as the main reference and OpenGeophy as supplemental reference.

+ When will the contents be updated?

Currently, OpenGeophy only contains courses materials from the geophysical engineering's undergraduate curriculum in Institut Teknologi Bandung, Indonesia. However, if you interested in adding to the content outside those, don't hesitate to contact us and join in developing OpenGeophy.

+ Found an error in the content?

OpenGeophy is very open to critics and suggestions in order to improve the quality of its contents. Thus, don't hesitate to contact us if you found any error in OpenGeophy's contents.
